# Product Context

## Why This Exists
AstroCalendar bridges ancient astrological wisdom with modern AI technology, offering users a personalized cosmic companion for daily guidance and self-reflection. It democratizes access to astrological insights traditionally requiring expensive consultations.

## Problems It Solves

### For Users
- **Lack of Personalization**: Generic horoscopes don't account for individual birth charts
- **Access Barriers**: Professional astrological readings are expensive and time-consuming
- **Language Barriers**: Most astrology apps lack proper localization for Turkish speakers
- **Consistency**: Daily engagement with self-reflection practices
- **Relationship Insights**: Understanding compatibility with partners/friends

### For the Market
- Combines both Western and Chinese astrological traditions in one app
- AI-generated insights provide unique, personalized content daily
- Offline-first architecture ensures reliability regardless of network conditions
- Beautiful, modern UI makes astrology accessible to younger audiences
- Multilingual support (English, Turkish, Thai) serves diverse global markets, with Thai as critical market (~60% of user base)

## How It Should Work

### User Flow
1. **Entry**: User discovers app â†’ sees welcoming cosmic interface
2. **Authentication**: Quick Google sign-in (no complex forms)
3. **Language**: Choose preferred language (affects all content)
4. **Onboarding**: Input birth date (required), time, place (optional)
5. **Profile Creation**: System computes zodiac signs and elements
6. **Dashboard**: Daily insight generated/retrieved, energy score displayed
7. **Exploration**: Browse calendar, view yearly forecast, check compatibility
8. **Retention**: Daily ritual reminders, new insights reinforce habit

### Key Interactions
- **Dashboard**: Central hub showing today's cosmic energy, lucky numbers, rituals
- **Calendar**: Swipe through dates to see past/future insights
- **Comparison**: Enter partner's birth date for compatibility reading
- **Profile**: View cosmic identity (zodiac signs, elements), manage settings

### Experience Goals
- **Mystical**: Dark theme, nebula effects, cosmic terminology ("Soul Sync", "Celestial Guide")
- **Personal**: Uses first name, speaks directly to user's astrological profile
- **Reliable**: Always shows something useful, even when AI unavailable
- **Delightful**: Smooth animations, satisfying interactions, beautiful typography
- **Informative**: Educational about zodiac systems while being entertaining

## Success Metrics
- Daily active usage (users checking morning insights)
- Engagement with rituals and recommendations
- Profile completion rate (adding birth time/location)
- Return visits within 24 hours
- Language preference distribution
